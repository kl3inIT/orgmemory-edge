# Build Week foundation design

Deliver a standalone vertical slice that demonstrates the product promise before native capture adapters are added:

- pilot starts/pauses a visible session;
- pilot selects local evidence and produces an SOP-shaped output;
- SOP flags human verification and can be exported as versioned JSON;
- admin sees device health but not raw context;
- browser demo and Tauri command paths share the same contract.

The seeded evidence is clearly development data, not a claim of live capture.
